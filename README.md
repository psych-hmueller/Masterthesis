# Masterthesis
Scripts for the master thesis "Suppression of intrasaccadic stimulus on- and offsets: active or passive mechanism?"

## Structure of the repository
`src`: Scripts for the statistical analysis of the study.

## Structure of the `src` folder:

`Datenbereinigung.R`: Script for cleaning and preparing the data. 

`Auswertung.R`: Main script for the statistical analysis. Also includes the graphs presented in the thesis.
